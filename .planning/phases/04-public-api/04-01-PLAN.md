Create lib/src/desktop_updater_api.dart as a new file with four top-level functions. All public members require /// doc comments (enforced by public_member_api_docs lint).

**checkForUpdate(UpdateSource source) → Future&lt;UpdateCheckResult&gt;**
- Wrap entire body in try-catch(Object e). On catch, throw NetworkError(message: "checkForUpdate failed: $e", cause: e)
- Call source.getLatestUpdateInfo() — if null, return const UpToDate()
- Call DesktopUpdaterPlatform.instance.getCurrentVersion() to get localBuild (int)
- If remoteInfo.buildNumber <= localBuild, return const UpToDate()
- Call generateLocalFileHashes() (from engine, already imported) to get localHashes
- Call source.getRemoteFileHashes(remoteInfo.remoteBaseUrl) to get remoteHashes
- Call diffFileHashes(localHashes, remoteHashes) to get changedFiles
- Return UpdateAvailable(remoteInfo.copyWith(changedFiles: changedFiles))

**downloadUpdate(UpdateInfo info, {void Function(UpdateProgress)? onProgress}) → Future&lt;void&gt;**
- Get appDir by resolving Platform.resolvedExecutable parent (macOS: two parents up to Contents/)
  Use: final appDir = File(Platform.resolvedExecutable).parent.parent.path (on macOS the staging goes under Contents/update/)
- Call downloadFiles(remoteBaseUrl: info.remoteBaseUrl, changedFiles: info.changedFiles, appDir: appDir)
- Listen to the stream; on each event call onProgress?.call(progress)
- Propagate stream errors by re-throwing (use await for ... completes naturally, stream errors become exceptions)

**applyUpdate() → Future&lt;void&gt;**
- try { await DesktopUpdaterPlatform.instance.restartApp(); }
- catch (e) { throw RestartFailed(message: "applyUpdate failed: $e", cause: e); }

**generateLocalFileHashes({String? path}) → Future&lt;List&lt;FileHash&gt;&gt;**
- Re-export: simply call and return the engine's generateLocalFileHashes(path: path)
- This is the public API surface wrapper — identical signature
